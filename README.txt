Notes:
- Works best on full-screen Chrome Browser
- The Timezone only changes when you manually change the timezone on the device you're running on
- The Location only changes when you manually change the location on the device you're running on
- The Zeppelins may not be at correct location if browser is not refreshed after browser size has been changed.

Build Instruction:
- Unzip the folder
- Open index.html on a browser (preferably Chrome)
